# [Valid Anagram - Easy](https://leetcode.com/problems/valid-anagram/submissions/)

```python
class Solution(object):
    def isAnagram(self, s, t):
        """
        :type s: str
        :type t: str
        :rtype: bool
        """

        return (
            len(s) == len(t)
            and 
            ''.join(sorted(s)) == ''.join(sorted())
        )
```
